HOW TO USE MAKEFILE
To run auto test use:
	make: compiles scheduler with normal goodness and without wait_in_rq, runs bush script with all the profiles
To compile normaly use:
	make normal: compiles scheduler with normal goodness
	make no_calc_wait: compiles without wait_in_rq
	
For auto test open jupiter notebook and run the python script after make. After that you can compare the results of the different tests thout plots.
